# basic-todolist-app

## Lakukan import database yang ada pada folder "db_dump"

## Project setup
```
npm install --save
```
## Compiles and run Node.js server
```
npm run start / nodemon
```
### "db_todolist" schema with structure
Data | Type | Size
------------ | -------------| -------------
id | int | 11
title | String | 100
date | String | 20
